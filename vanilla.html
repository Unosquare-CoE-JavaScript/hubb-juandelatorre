Algorithm 1 - Regular Expressions

This algorithm uses regular expressions to, first, escape groups of numbers: replace(/(\d+)/g, '*$1') After that, groups of 3 or more identical characters are selected with the following regular expression: /([a-z])\1{2,}/ig And replaced with the character followed by the number of repetitions: match => match[0] + match.length

Algorithm 2 - Array Functions

This algorithm uses a reduce function to analyze the string, identify groups of numbers and groups of identical characters, and save this information in an array of intermediate values: { char: 'a', count: 3, isNumber: false } Then it maps each group to a string. First escaping numbers: isNumber ? `*${char}` : ... And, then, filtering string longer than 2: ... count > 2 ? `${char}${count}` : char.repeat(count)
